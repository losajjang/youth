# PRODUCT_SPEC

## One-line

사용자의 청춘 시절에 맞는 기사와 영상 콘텐츠를 피드 형태로 보여주는 모바일 앱

## Target users

- 30~50대 사용자
- 추억, 회상, 향수 콘텐츠를 좋아하는 사용자

## Core value

- 검색하지 않아도 청춘 시절의 콘텐츠를 발견하게 한다
- 영상뿐 아니라 기사, 블로그, 아카이브 링크도 함께 보여준다
- 세대와 시절 중심으로 콘텐츠를 큐레이션한다

## MVP

1. 온보딩에서 청춘 시절 연도 범위 선택
2. 홈 피드에서 콘텐츠 카드 목록 조회
3. 좋아요 / 저장
4. 저장 목록 조회
5. 마이페이지

## Out of scope

- 댓글
- 사용자 업로드
- 복잡한 추천 알고리즘
- 관리자 웹
- 결제 기능

## UI direction

- iOS 느낌의 단정한 카드 UI
- 큰 썸네일
- 짧은 제목
- 최소한의 액션 버튼

## Data source (initial)

- 더미데이터
- 이후 YouTube / 기사 링크 등으로 확장

## Success criteria

- 앱 첫 실행 후 1분 안에 첫 콘텐츠를 볼 수 있다
- 온보딩부터 피드 진입 흐름이 자연스럽다
